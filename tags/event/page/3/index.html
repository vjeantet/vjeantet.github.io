Docker.io Paris MEETUP #5

Veille de saint Valentin 2014, les curieux, passionnés et amoureux de Docker se sont rencontrés encore une fois pour échanger leurs expériences autour de Docker.io Docker n.m. (Début XXIe). La startup Nuxeo nous a accueilli dans ses locaux juste après le boulot, avec tout le matériel nécessaire, bonne humeur, bière, sodas, chips etc.. GOlang, chez Google

Il fallait y être ce soir : liste d’attente, contrôle sécurité, signalétique android, salles ultramodernes, bref c’était dans les locaux de Google France, une attraction rien qu’à eux ! »
